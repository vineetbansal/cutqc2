class cutqc2.core.dag.DagNode(wire_index, gate_index, name='q')[source]¶

Bases: object

Represents a node in a quantum circuit DAG (Directed Acyclic Graph), corresponding to a specific gate on a specific wire (qubit/register).

Since only inter-wire gates are important for the cut algorithm, any mention of gate_index refers to the index wrt inter-wire gates only.

wire_index¶

The index of the wire (qubit/register).

Type:

int

gate_index¶

The index of the gate on the wire. Note: gate_index assumes that only inter-wire gates are considered.

Type:

int

cutqc2.core.utils.distribute(specs, capacity=None, zoom=None)[source]¶

Distribute (or redistribute) loads to fit within a specified total capacity. :type specs: dict[int, str] :param specs:

  • “A”: qubit is preserved in output

  • “M”: qubit is summed over

  • “0”/”1”: qubit is fixed to that value

Parameters:

  • capacity (Total capacity to distribute loads across.)

  • zoom (Optional id to zoom into)

Return type:

dict[int, str]

Returns:

  • Dictionary mapping id to new load spec

  • New load spec distributes the available capacity to each id’s ‘M’ qubits

  • by changing a subset of them to ‘A’.
